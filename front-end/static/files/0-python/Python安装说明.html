Python是一种面向对象的解释型编程语言。Python官方提供的解释器是CPython，它是跨平台的，可以运行在Windows、Mac和各种Linux/Unix系统上。

为了学习Python编程，首先需要安装Python相关的一些程序。安装后，你会得到Python解释器（就是负责运行Python代码的程序），和一个简单的集成开发环境。

0. Python官网

Python的官网是https://www.python.org/，在这里可以找到所需的程序下载、文档等各种资料。一般情况下，应该使用包管理软件或者从官网下载的安装包来安装Python。

1. 版本选择

Python目前有2.x和3.x两个版本，它们是语法相似的不同语言，在官网有一份它们的比较。本次讲座以Python 3.x为基础。

2. 安装Python

2.0 Debian / Ubuntu

以root用户运行apt-get install python3即可（非root用户在指令前加上sudo）。安装完成后，打开命令行，运行输入python3，如果出现下面的内容说明安装成功了

Python 3.5.2 (default, Sep 10 2016, 08:21:44)
[GCC 5.4.0 20160609] on linux
Type "help", "copyright", "credits" or "license" for more information.
>>> _

2.1 Windows (Native)

从官网直接下载最新版的安装程序，并安装即可。安装完成后，打开命令行（先按Win+X，再按C），输入python，出现类似于上面的内容说明安装成功了

可以输入exit()并按回车来退出。

2.2 Windows (cygwin)

运行setup-x86.exe或setup-x86_64.exe，搜索并安装python3即可。打开cygwin的terminal，运行python3，看到类似上面的信息即表示安装成功。

2.3 其他

对于有包管理或官网有安装包的系统，参照相应方式进行安装。

自行编译的方法如下：

wget https://www.python.org/ftp/python/3.5.2/Python-3.5.2.tgz
tar xf Python-3.* 
cd Python-3.*
./configure
make
make install

3. 安装IDE

部分安装包中在安装python之后会提供一个名为IDLE的工具，这是一个十分简陋的IDE，我们可以使用它来进行开发。

实际上，我们也可以使用任意的现代化的编辑器（notepad++/sublime/emacs/vim/…）来编写python源文件，将其存储（一般是以.py或.pyw做为后缀）后用python3 foo.py来执行它。

另外，我个人推荐使用JetBrains公司的PyCharm来进行Python开发。作为JB系IDE，它具有完善的开发、调试功能，也能够很好处理多文件的工程。它同样支持多种操作系统。它可以从官网上下载，用edu邮箱（如清华邮箱）在官网上注册一个学生账户即可免费使用JetBrains全系列IDE的professional版本，包括我们需要的PyCharm。

同时，Visual Studio配合使用Python Tools for Visual Studio (PTVS)也可以十分方便地进行Python开发和调试。如果大家对VS较为熟悉，可以尝试使用它。
