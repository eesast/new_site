TUNet web版协议

在登录过程中打开浏览器的网络调试页（F12）就可以看到请求的情况。

登录

登录服务器需要用到明文用户名和密码的MD5值，密码的MD5值可以通过下面的方法获取：

import hashlib
ret = hashlib.md5('<your_password>'.encode('utf-8')).hexdigest()

POST的内容包括

  • action：'login'
  • user_name：用户名
  • user_password：'{MD5_HEX}' + 密码的MD5值
  • ac_id：常量1

将它们作为POST数据，请求https://net.tsinghua.edu.cn/do_login.php，例如我们声明一个：

login_dict = {'action':'login', 'user_name':'wqr14', 'user_password':'{MD5_HEX}5f4dcc3b5aa765d61d8327deb882cf99', 'ac_id':1}

然后发送该请求：

import requests
ret = requests.post('https://net.tsinghua.edu.cn/do_login.php', data=login_dict).text

返回’login_ok’说明登录成功

下线

与登录类似，不同的地方是POST的内容

  • action：'logout'

返回’logout_ok’说明下线成功

查询在线状态

与登录类似，不同的地方是POST的内容

  • action：'check_online'

返回’not_online’说明不在线，返回’online’说明在线

在线时，我们可以进一步发送GET请求https://net.tsinghua.edu.cn/rad_user_info.php（不需要参数）来获取在线用户信息：

import requests
ret = requests.get('https://net.tsinghua.edu.cn/rad_user_info.php').text

返回信息的格式是：

用户名, 登录时间, 服务器时间, 未使用, 未使用, 未使用, 未使用, 未使用, 当前IP, 未使用, 未使用, 帐户余额, 未使用

登录时间和服务器时间是UNIX timestamp，我们可以将它转换成便于阅读的形式：

import time
time_str = time.ctime(unix_timestamp)
